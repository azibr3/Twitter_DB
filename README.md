# Twitter Database System

A relational database project that simulates a simplified version of Twitter, built as a final project for SQL 101–103 on the Satr Data Analysis Track.

## What I practiced
- Designing a relational database with multiple tables
- Using Foreign Keys to maintain data integrity
- Building stored procedures for account creation and follow system
- Password hashing using MD5 + UNHEX
- Writing aggregation queries

## Tools
- MySQL
- MySQL Workbench

## About
This project is part of the Satr Data Analysis Track (SQL 101–103 final project).
